Ruby vs Javascript going loopy

17th January 2015

We have changed tack and started learning Javascript this week.

1. Looping in Ruby

So, if we have a simple array, we can loop through it using the Ruby enumerable each merely telling it to take each item in the array and add it to our sum. Yields the result 6. 


my_array = [1,2,3]
sum = 0 
my_array.each { |x| sum +=x  }
p sum //6

2. Looping in Javascript

In contrast, javascript does not have an each builtin method (underscore is a library that can access an each function - but it is not built-in). In order to loop through the array we need to tell the interpreter 2 key things: 1. To create a variable "i", start at 0, continue looping through to the array til the final index (i < array.length) and continue going upwards through the array items by increasing index (i++). 2. We then need to tell it to add each item in the array (my_array[i]) to the result variable. This too yields our simple result of 6. But with a bit more work and typing !! 


var my_array = [1,2,3];
var result = 0;
for(var i=0; i < my_array.length; i++) {result += my_array[i];}
console.log(result); //6
